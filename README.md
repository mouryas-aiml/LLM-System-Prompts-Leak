# LLM-System-Prompts-Leak

Collection of **LLM System Prompts, Developer Messages, and Prompt Injection Examples** gathered for **educational purposes and AI security research**.

---

## About This Repository

This repository documents publicly discovered **system prompts used in Large Language Models (LLMs)**.

System prompts control how AI models behave, enforce safety rules, and guide responses.  
Some of these prompts have been exposed through **prompt injection experiments, research discussions, and public documentation**.

This archive helps researchers and developers study:

- LLM prompt architecture
- Prompt injection techniques
- AI safety guardrails
- Prompt engineering strategies
- AI transparency

---

## Repository Contents

- **System Prompts** from different AI platforms
- **Prompt Injection Examples**
- **Developer Message Structures**
- **Prompt Security Research Notes**
- **AI Safety References**

---

## Disclaimer

This repository is maintained **strictly for educational and research purposes**.

All prompts included here were **publicly discovered, discussed, or shared in open communities**.

This project **does not encourage hacking, misuse, or exploitation of AI systems**.  
Its goal is to help the AI community better understand **prompt engineering, AI security, and system prompt design**.

If any platform owner wishes to remove specific content, please open an issue.
